## 🔧 HƯỚNG DẪN CHUYỂN ĐỔI SANG WORD/PPTX

### Từ Markdown → Word
1. Mở file .md bằng VS Code
2. Cài extension "Markdown PDF" hoặc "Markdown All in One"
3. Export sang PDF hoặc copy nội dung vào Word
4. Hoặc dùng Pandoc: `pandoc file.md -o file.docx`

### Từ Y5 → PowerPoint
1. Đọc nội dung từng slide trong Y5_Slide_Scrum.md
2. Tạo PowerPoint mới với template professaional
3. Copy nội dung từng slide
4. Thêm hình ảnh, screenshots, diagrams
5. Áp dụng theme dark (#1E1E2E) + gradient accent

### Sơ đồ phân rã → Visio
1. Mở Microsoft Visio
2. Chọn template "Organization Chart" hoặc "Block Diagram"
3. Tham khảo cấu trúc trong Y1 (phần Mermaid diagram)
4. Vẽ các node theo hierarchy 3 cấp
